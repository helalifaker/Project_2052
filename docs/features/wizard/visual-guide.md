## 14. Accessibility Features

- **Keyboard Navigation:** All buttons and cards focusable
- **Focus Indicators:** 2px ring on focus
- **ARIA Labels:**
  - `aria-label="Edit Proposal Basics"`
  - `aria-label="Expand"` / `aria-label="Collapse"`
  - `role="alert"` on error states
  - `aria-live="assertive"` on error states
- **Screen Reader:** Error messages announced immediately
- **Color Contrast:** All text meets WCAG AA (4.5:1 minimum)
